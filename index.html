Conectar tu tienda

Ingresa tus credenciales de la API Admin de Shopify para ver tus variantes mas vendidas en tiempo real.

  1. Ve a tu panel de Shopify → Configuracion → Aplicaciones y canales de ventas
  2. Haz clic en Desarrollar aplicaciones → Crear una aplicacion
  3. En permisos de Admin API, habilita: read_orders, read_products
  4. Instala la app y copia el Token de acceso
  5. Tu dominio es: tu-tienda.myshopify.com
⚠️ Nota CORS: Los navegadores bloquean peticiones directas a la API de Shopify. Esta app usa corsproxy.io como intermediario seguro. Si tienes bloqueadores de contenido, desactivarlos puede ser necesario.

📊 Proyeccion de Pedidos
📌 Notas importantes

% Devolucion: Corresponde al porcentaje de ordenes devueltas sobre las despachadas (ya restando cancelados y rechazados que pueden sumar un 10% adicional).

Control diario: Si las devoluciones son del 25%, la inefectividad en tu control diario debe ser del 35%, midiendo con un 65% de efectividad.

Consideracion: Usar la calculadora sin llevar el control diario es imposible.

Formulas:
Flete con devol. = Flete base / (1 − %Devolucion)
CPA 20% = (Proveedor + Flete devol. + Admin + Fullfilment + Otros) / 0.6 × 20%
Costos totales = Suma costos + MAX(CPA 20%, CPA base)
Utilidad 20% = Costos totales / 0.8 × 20%
Precio de venta = Costos totales + Utilidad 20%
